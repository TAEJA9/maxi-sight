# Skills-C.md · 시각화 선택 기준

Maxi-Sight 분석 파이프라인 Module C — Visualization Rules


## 목적

Module B에서 계산된 지표값을 어떤 차트·컴포넌트 유형으로 렌더링할지 결정하는 규칙을 정의한다. AI는 이 규칙을 참조하여 데이터 특성에 맞는 시각화 유형을 자동으로 선택한다.


## 1. 차트 유형 선택 매트릭스

| 데이터 유형 | 차트 유형 | 사용 뷰 | 선택 이유 |
|---|---|---|---|
| 자산군별 배분 비중 | 도넛 차트 | V3 | 구성 비율 파악에 최적, 전체 합이 100%임을 직관적으로 표현 |
| 종목별 평가액 집중도 | 트리맵 | V6 | 면적으로 비중 표현, 종목 수가 많을 때 한눈에 분포 파악 |
| 수익률 시계열 | 라인 차트 | V4 | 추세 흐름과 벤치마크 오버레이 동시 표현 가능 |
| 종목별 수익 기여도 | 수평 바 차트 | V4 보조 | 종목 간 크기 비교가 직관적, 종목명 레이블 가독성 우수 |
| 상관관계 매트릭스 | 히트맵 | 선택적 | 다차원 관계를 색상 농도로 한눈에 파악 |
| 리스크-수익률 분포 | 산점도 | 선택적 | 효율적 투자선(Efficient Frontier) 시각화 |
| 단일 수치 KPI | 숫자 카드 + SVG 아크 | V2 | 핵심 지표 3개(CAGR·Sharpe·MDD)를 단독 강조, 0-100 점수 아크 병기 |
| 보조 KPI 6종 | 소형 숫자 카드 | V2 보조 | Sortino·베타·변동성·연 배당·월 배당·배당 종목 수 |
| 총 자산 현황 | 요약 카드 | V1 | 숫자 + 변동액 + 변동률 세트로 표시 |
| AI 포트폴리오 건강도 | 반원 게이지 + 레이더 | V5A | 5개 차원 종합 점수, 차원별 진행 바 + 레이더 차트 |
| AI 인사이트 | 텍스트 카드 목록 | V5B | 자연어 문장 출력, 트리거 유형별 색상 구분, 필터 탭 제공 |
| 뉴스 감성 점수 | 감성 배지 | V2 보조 | 종목 카드 옆 긍정/중립/부정 색상 배지 |


## 2. 뷰별 상세 렌더링 규칙

### GNB · 글로벌 내비게이션 바

- sticky 헤더, 백드롭 블러 적용
- **좌**: 로고(Activity 아이콘 + "Maxi-Sight by MaxItOut")
- **중앙**: 탭 3개 — 오버뷰 / 포트폴리오 / 인사이트 (클릭 시 해당 섹션으로 스무스 스크롤)
- **우**: 다크/라이트 테마 토글 · 새로고침(스피너 애니메이션) · 페르소나 드롭다운(User 아이콘)

### PersonaInfoBar · 페르소나 정보 바

- 각 섹션 최상단에 글래스모피즘 카드로 배치
- **좌**: 총 수익률(%) 대형 표시 + 수익/손실 금액 + 총 자산 평가액
- **우**: 연령대 배지 · 이름 · 투자 스타일 태그 · 아바타(그라디언트 + 온라인 도트)
- 수익률 색상 기준: 상승 빨강, 하락 파랑 (국내 컨벤션 동일)
- 페르소나별 아바타 그라디언트 색상 고정 (portfolio_id 기준 매핑)

### V1 · 종합 잔고 허브

- 총 자산 평가액을 **원화(KRW)** 기본 표시, 달러(USD) 병렬 표시

- 전일 대비 변동액: `+/- N원` 형식, 상승 시 빨간색, 하락 시 파란색 (국내 컨벤션)

- 전일 대비 변동률: `+/- N.NN%` 형식

- 계좌별 소계를 접이식(Accordion) 형태로 제공

**계좌 표시 목록 (7종 전체)**

| 코드 | 표시명 | 비고 |
|---|---|---|
| 01 ISA | ISA | 비과세 한도 잔여 병기 가능 |
| 02 IRP | IRP | 과세이연 배지 표시 |
| 03 GEN | 일반 위탁 | — |
| 04 OVS | 해외 주식 | USD 병렬 표시 |
| 05 PEN | 연금저축 | 과세이연 배지 표시 |
| 06 CMA | CMA/파킹 | 이자 수익 별도 표기 |
| 07 CRP | 가상자산 | USD 병렬 표시, 변동성 경고 배지 가능 |

- 과세이연 계좌(IRP, 연금저축)에는 🔒 배지 표시

- 가상자산 계좌는 24시간 기준 변동률 표시 (주식과 구분)

### V2 · 3대 핵심 지표 카드

- **메인 KPI 3개** (가로 3열, 모바일: 세로 스택)

- 각 카드 구성:
  - 상단: 지표명 + 이지가이드 아이콘 + 부제(예: "연평균 성장률")
  - 중앙: 수치(대형) + SVG 원형 점수 아크 (0~100 정규화)
  - 하단: Module E 번역 문구 한 줄 (회색 소형 텍스트)
  - 배경: 지표 상태(긍/주/위)에 따른 accent 색상 그라디언트

- 수치·아크 색상 기준:

| 지표 | 긍정(Accent 색) | 주의 | 위험 |
|---|---|---|---|
| CAGR | ≥ 7% → emerald | 3~7% → amber | < 3% → red |
| Sharpe | ≥ 1.0 → indigo | 0.5~1.0 → amber | < 0.5 → red |
| MDD | < 5% → emerald | 5~15% → amber | ≥ 15% → red |

- **보조 KPI 6개** (가로 6열, 모바일: 3×2 그리드)

| 레이블 | 필드 | 부제 |
|---|---|---|
| Sortino | `sortino_ratio` | 하락장 맷집 |
| 베타 | `portfolio_beta` | 시장 민감도 |
| 변동성 | `volatility_annualized_pct` | 연환산 스트레스 |
| 연 배당 | `dividend.annual_dividend_krw` | 예상 연간 배당 |
| 월 배당 | `dividend.monthly_dividend_krw` | 월평균 수령 예상 |
| 배당 종목 | `dividend.dividend_items.length` | 배당 지급 종목 수 |

- 배당 데이터 없을 때: 해당 셀 `—` 표시

### V3 · 포트폴리오 배분 차트

- 도넛 차트 중앙에 총 자산 평가액 표시

- 자산군 범례: 코드 대신 한글명 표시

| 코드 | 범례 표시명 |
|---|---|
| EQ | 국내 주식 |
| EQ-F | 해외 주식 |
| ET | 국내 ETF |
| ET-F | 해외 ETF |
| BD | 채권 |
| CS | 현금/CMA |
| CM | 금/원자재 |
| CR | 가상자산 |
| SV | 저축/청약 |

- **목표 배분 비교**: 사용자 설정 목표 비중과 현재 비중을 차트 우측에 나란히 표시

- 이탈 범위(±5%p 초과) 항목은 강조 표시 (굵은 테두리 + 경고 아이콘)

- 미설정 시: 목표 배분 비교 영역 숨김

### V4 · 수익률 타임라인

- X축: 날짜 (기간 선택 탭: 1M / 3M / 6M / 1Y / 전체)

- Y축: 수익률(%) — 절대값이 아닌 기준일 대비 상대 수익률

- 라인 구성:

- **내 포트폴리오**: 굵은 실선 (주색)

- **KOSPI**: 얇은 점선 (회색)

- **S&P 500**: 얇은 점선 (연한 파랑)

- 특정 날짜 클릭 시 당일 평가액·수익률·벤치마크 대비 초과수익 툴팁 표시

- 가상자산(CR) 포함 포트폴리오는 주말·공휴일 데이터 포함 (24시간 거래 자산)

### V5A · AI 포트폴리오 건강도 카드

- V5InsightFeed 위에 별도 glass-card로 배치

- **구성**:
  1. 헤더: Brain 아이콘 + "AI 포트폴리오 분석" + "5개 지표 종합 평가" + 분석 상태(로딩/완료 dot)
  2. 반원 게이지(HealthGauge) — 0~100점, 점수별 색상: ≥70 emerald / 40~69 amber / <40 red, 등급 표시(양호/보통/주의)
  3. 5개 차원 진행 바:

| 차원 | 산출 기준 |
|---|---|
| 수익성 | CAGR / 15% 정규화 (max 100) |
| 효율성 | Sharpe / 2.0 정규화 |
| 리스크 | 100 - (MDD / 30) × 100 |
| 안정성 | 100 - (변동성 / 35) × 100 |
| 분산도 | (1 - HHI) × 115, 허핀달 지수 기반 |

  4. 레이더 차트 — 5차원 값을 레이더(spider) 형태로 표시

- **건강도 종합 점수** = 5개 차원 단순 평균 (0~100 정수)

### V5B · AI 인사이트 피드

- 카드 목록 형태, 우선순위(HIGH→MEDIUM→LOW) + 최신순 정렬

- **필터 탭** (피드 상단):

| 탭 | 포함 유형 |
|---|---|
| 전체 | 모든 카드 |
| ⚠️ 위험 | 위험 알림, 변동성 경보 |
| 🔄 리밸런싱 | 리밸런싱 알림, 집중 위험 |
| ✅ 기회 | 기회 신호 |
| 💡 정보 | 정보 알림, 뉴스 감성 경보 |

- 카드 유형별 좌측 테두리 색상 구분:

| 트리거 유형 | 색상 | 아이콘 |
|---|---|---|
| 위험 알림 | 빨강 테두리 | ⚠️ |
| 리밸런싱 알림 | 주황 테두리 | 🔄 |
| 기회 신호 | 초록 테두리 | ✅ |
| 집중 위험 | 노랑 테두리 | 📍 |
| 변동성 경보 | 빨강 테두리 | 📈 |
| 뉴스 감성 경보 | 보라 테두리 | 📰 |
| 정보 알림 | 회색 테두리 | 💡 |

- 카드 우선순위 배지: 긴급(빨강) / 주의(노랑) / 참고(회색)
- AI 생성 카드에 `✦ AI` 배지 표시 (`ai_generated: true` 시)
- 카드 하단: 생성 시각 + 생성 출처 (예: `오늘 오전 9:32 · Gemini AI`)
- 카드 4개 초과 시 하단 페이드 그라디언트 + 스크롤

### V6 · 종목별 트리맵

- 보유 종목을 평가액 비례 면적으로 표시 (Recharts `<Treemap>`)
- 자산 클래스별 색상 매핑:

| 코드 | 색상 |
|---|---|
| EQ | blue (#3b82f6) |
| EQ-F | indigo (#6366f1) |
| ET | emerald (#10b981) |
| ET-F | green (#059669) |
| BD | amber (#f59e0b) |
| CS | gray (#6b7280) |
| CM | orange (#d97706) |
| CR | purple (#a855f7) |
| SV | sky (#0ea5e9) |

- 셀 크기가 충분할 때 (width > 60, height > 40): 종목명 + 평가액 텍스트 표시
- 호버 툴팁: 종목명 · ticker · 평가액(원화 포맷)
- V3(도넛)과 동일 행에 배치: `grid-cols-[380px_1fr]`


## 3. 이지가이드 팝업 규칙

- 금융 용어에 `?` 아이콘 부착, 호버(PC) / 탭(모바일) 시 팝업 표시

- 팝업 내용은 Skills-E.md의 용어 번역 테이블에서 자동 조회

- 팝업 위치: 해당 요소 기준 상단 우선, 뷰포트 초과 시 하단으로 자동 전환


## 4. 반응형 브레이크포인트

| 구간 | 기준 | 레이아웃 변화 |
|---|---|---|
| 데스크톱 | ≥ 1024px | V1~V5 전체 그리드 표시 |
| 태블릿 | 768~1023px | V3·V4 2열 → 1열 전환 |
| 모바일 | < 768px | 모든 뷰 단일 컬럼, V2 카드 세로 스택 |


## 5. 차트 공통 스타일 규칙

- 폰트: 시스템 기본 산세리프 (Pretendard 우선, 없으면 Apple SD Gothic Neo / Noto Sans KR)

- 색상 팔레트: 최대 8색 사용 (자산 클래스 8종 대응), 색각 이상자 고려 (색상+패턴 병행)

- 애니메이션: 차트 초기 로딩 시 0.4초 페이드인, 데이터 갱신 시 0.2초 전환

- 빈 데이터 상태: 스켈레톤 UI 표시 (스피너 사용 금지)
